Story embedding space — coloured by genre

Each dot is one story. Position encodes structural similarity (pacing, emotional arc, discourse rhythm) — not topic or lexical similarity. Stories that "feel" structurally alike land close together.

Story embedding space — coloured by HDBSCAN cluster

Clusters found unsupervised. Genre boundaries and cluster boundaries partially overlap — that overlap is the hypothesis signal.

Genre archetypes — mean normalised feature trajectory per cluster

Each feature is min-max normalised to [0,1] per story before averaging, so all features share the same y-axis. White line = composite "Shape of Story" arc.

Author clustering

Convex hulls show each author's spread in story space. Purity = fraction of that author's stories landing in their dominant cluster.

Hypothesis test — Classical vs Full+EB features

Classical = sentence-level + syntactic + affective + discourse + information + sequential features only (language-agnostic). Full = Classical + SBERT PCA-50 embeddings. Higher ARI for Classical confirms that statistical discourse features alone structure genre.
